# 🎮 Java Number Guessing Game

A simple **Java console-based Number Guessing Game** developed using **Core Java**. This project is ideal for beginners to understand Java programming concepts such as loops, conditionals, random number generation, methods, and user input handling.

---

## 📌 Project Overview

The Number Guessing Game generates a random number within a specified range. The player attempts to guess the correct number, and the program provides hints whether the guess is **too high** or **too low** until the correct number is found.

This project was created as my **first Java programming project** to strengthen problem-solving skills and understand the fundamentals of Java.

---

## ✨ Features

* 🎯 Random number generation
* 🎮 Interactive gameplay
* 💡 Hints after every incorrect guess
* 📊 Tracks the total number of attempts
* ✅ Displays a winning message when guessed correctly
* 🖥️ Simple command-line interface
* 🚀 Beginner-friendly source code

## ▶️ How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/Java-Number-Guessing-Game.git
```

### 2. Navigate to the Project Folder

```bash
cd Java-Number-Guessing-Game
```

### 3. Compile the Program

```bash
javac NumberGuessingGame.java
```

### 4. Run the Program

```bash
java NumberGuessingGame
```
